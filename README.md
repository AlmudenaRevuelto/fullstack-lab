# Fullstack Lab 🧪

This repository contains exercises, experiments and small projects
focused on improving my full-stack development skills and preparing
for technical interviews.

## Structure
- `javascript/` – JS fundamentals, async patterns, algorithms
- `php/` – Backend logic, APIs, WordPress-style development
- `java/` – OOP, collections, business logic
- `python/` – Scripts, backend logic, data handling
- `sql/` – Queries, schema design, optimization

## Goal
To maintain a consistent practice routine and showcase real coding skills.
